# 🤖 Rule-Based AI Chatbot

## 📌 Project Overview

This project is a simple Rule-Based AI Chatbot developed using Python. The chatbot interacts with users by responding to predefined inputs using conditional statements (`if-elif-else`) and runs continuously until the user exits.

## 🎯 Objective

To understand the fundamentals of Artificial Intelligence by creating a chatbot that can:

* Handle greetings
* Respond to predefined questions
* Accept user input
* Run continuously in a loop
* Exit gracefully when requested

## 🛠 Technologies Used

* Python 3
* Visual Studio Code (VS Code)
* Git & GitHub

## ✨ Features

* Responds to greetings such as:

  * Hello
  * Hi
  * Hey

* Answers simple questions:

  * How are you?
  * What is your name?

* Handles exit commands:

  * Bye
  * Exit
  * Quit

* Uses:

  * `if-elif-else` decision making
  * `while True` loop
  * `break` statement

## 🚀 How to Run

1. Open the project folder in VS Code.
2. Open the terminal.
3. Run:

```bash
python chatbot.py
```

## 💬 Example Conversation

```text
🤖 Welcome! I am a Rule-Based AI Chatbot.

You: hello
Bot: Hi there!

You: how are you
Bot: I am fine. Thanks for asking!

You: bye
Bot: Goodbye! Have a nice day.
```
